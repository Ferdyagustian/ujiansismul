# Galeri Kenangan (Canvas Memori Digital)

Aplikasi "Galeri Kenangan" adalah sebuah sistem manajemen konten (CRUD) untuk mendokumentasikan momen-momen penting. Aplikasi ini dibangun dengan struktur Model-View-Controller (MVC) menggunakan framework CodeIgniter 3. 

## Tech Stack
* Backend: PHP 7.x/8.x dengan Framework CodeIgniter 3
* Database: MySQL (MariaDB)
* Frontend: HTML5, CSS3 (termasuk custom CSS pada assets/css/style.css), dan JavaScript
* Server: XAMPP (Apache)

## Keterangan Program
Pada versi pembaruan ini, aplikasi mengusung tema "Canvas Memori Digital" dengan desain UI cerah (light mode) berlatar grid modern. Aplikasi memiliki halaman "Beranda" (Home) sebagai pusat kendali untuk melihat ringkasan (Highlights & Snapshots) jumlah kenangan dan foto, serta menyediakan Quick Actions untuk aksi cepat.

## Fitur Utama
1. Beranda / Dashboard: Menampilkan statistik ringkas (Total Kenangan, Total Foto, Kategori Aktif) dan menu aksi cepat (Quick Actions).
2. Daftar Kenangan (Read): Menampilkan seluruh data kenangan dalam bentuk *card* galeri.
3. Tambah Kenangan (Create): Form popup untuk memasukkan data kenangan dengan dukungan multiple upload foto sekaligus.
4. Detail Kenangan: Menampilkan data kenangan dan seluruh lampiran foto.
5. Edit Kenangan (Update): Memperbarui informasi teks dan menambah/menghapus foto parsial.
6. Hapus Kenangan (Delete): Menghapus data kenangan berserta file foto fisik dari server secara otomatis.
7. Halaman Anggota Kelompok: Menampilkan identitas pengembang.

## Struktur Database (`database_update.sql`)
Database: kenangan
1. tb_kenangan
   - id_kenangan (INT, Primary Key, Auto Increment)
   - judul (VARCHAR)
   - kategori (VARCHAR)
   - tanggal_momen (DATE)
   - deskripsi (TEXT)
2. tb_foto
   - id_foto (INT, Primary Key, Auto Increment)
   - id_kenangan (INT, Foreign Key)
   - nama_file (VARCHAR)

## Panduan Instalasi (Lokal)
1. Aktifkan Apache & MySQL di XAMPP.
2. Buat database baru bernama `kenangan` di phpMyAdmin.
3. Import file `database_update.sql` ke database tersebut.
4. Akses aplikasi melalui tautan dinamis yang telah disediakan (http://localhost/ujiansismul/).
